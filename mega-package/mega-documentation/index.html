Main File Structure

All the file are well organized, its so easy to work with the template.
1. Unzip the files.
2. Open "Mega Template" folder.
You will find the index.html file on the root that will be edited.
Also you will see 4 directories, css, js, fonts and images in asset folder.
In the first folder Content only styling files the all the other files expect the htmls.
Js folder contain only the jquery library. Fonts folder contain essetial font files for fonts.

HTML5 Builder

Mega HTML5 template includes awesome page builder. You can generate your site, change content with any design. We have included 95+ carefully developed html5 blocks you can use.

It is very easy to install builder. On your download file, you will get a folder called meta-html5-builder. Upload that folder content on your server. You could upload those in yoursite.com/builder location. It can be used in localhost too. Just make sure your server has php 5.4 or above.

After upload those files, go go yoursite.com/builder. You will see builder loads & you can use blocks by using drag & drop. If you click on any content location, it will open editor where you can change content. After build your site, you can generate zip as export.
